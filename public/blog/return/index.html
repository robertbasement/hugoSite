年底去熊本要怎麼規劃哩

日期 : 2025/12/25 TPE -> KMJ, 2025/12/28 KMJ -> TPE

由於是12/25 聖誕節一早就到熊本, 12/28一早就離開。
實際能遊玩的時間是 25, 26, 27 組共3天。

推薦怎麼樣都要去的景點 -> 阿蘇火山
建議是要租車前往?

如果阿蘇我們只是要看破火山口 可以一日遊
沒有開車的話.
熊本-> 阿蘇交通彙整 JR熊本站 -> (JR)阿蘇站【時間：約80分鐘/費用：¥1880】 ->(巴士) 阿蘇山上總站【時間：約35分鐘/費用：¥730】 ->(巴士) 阿蘇火山口遊客中心【時間：約5分鐘/費用：¥600】.

可以第一天 熊本市區 水前寺 +

第二天(整天) 阿蘇火山(中岳, 破火山口) + 草千里
可以考慮騎馬, 吃冰淇淋

第三天 熊本城 + 櫻之馬場(團子)
